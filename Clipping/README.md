# Cắt hình theo view

Hàm truyền vào (x1,y1),(x2,y2) là tọa độ điểm trên trái và dưới phải của hình chữ nhật, (xa,ya),(xb,yb) là hai đầu đoạn thẳng cần vẽ.\
